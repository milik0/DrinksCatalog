About Drinks Catalog

Our Mission

Drinks Catalog is a web application built with Flask and MongoDB that allows bars to manage their catalog of drinks. It provides a simple and intuitive interface to add, view, edit, and delete drinks from the catalog.

The application features a user authentication system that allows bars to create an account and manage their own catalog of drinks. This makes it easy for bars to keep track of their inventory and update their drink offerings as needed.

Our System

The drink model used in the application is composed of a name and a type, which indicates whether the drink is alcoholic or not. This simple model allows bars to easily categorize and filter their drinks based on their needs.

The application uses Flask as the web framework and MongoDB as the database. Flask provides a simple and flexible way to build web applications, while MongoDB provides a scalable and performant way to store and retrieve data.
